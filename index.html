Video Speed Controller

Drag the button below to your browser's Bookmarks Bar.
Then, on any page with a video, click it to set your desired speed!

► Speed Control

How to Use This

  1. Make your Bookmarks Bar visible (Ctrl+Shift+B or Cmd+Shift+B).
  2. Drag the "► Speed Control" button above to your Bookmarks Bar.
  3. Visit any page with a video (YouTube, Netflix, etc.).
  4. Click the bookmark. A prompt will appear to set the speed.

Note: This page must be hosted on a web server (like GitHub Pages) to work correctly. It will not work from a local file.
